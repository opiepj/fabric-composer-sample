Business Network (Blockchain Backend)

Defining a business network is the entry point to any project. The business network contains participants, assets, and transactions. This represents the interface for the Fabric blockchain or backend.

Composer Module (Middle Layer)

The Fabric Composer module is used to provide logic for handling asset and transaction management for your particular business networks. All the logic can be spun up on an express middle layer server that client applications can interact with through standard REST interactions.
